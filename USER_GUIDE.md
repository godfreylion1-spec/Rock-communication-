# ROCK User Guide

Welcome to ROCK - the premium social messaging platform. This guide will help you get started and make the most of all features.

## Getting Started

### 1. Create Your Account

1. Visit the ROCK app
2. Click "Get Started with ROCK"
3. Sign in with your account
4. Complete your profile setup

### 2. Set Up Your Profile

1. Upload a profile picture (avatar)
2. Add your display name
3. Write a short bio
4. Set your status (available, away, etc.)

### 3. Find Friends

1. Go to **Contacts**
2. Search for friends by name or username
3. Send friend requests
4. Accept incoming friend requests

## Features Guide

### Messages

#### Sending Messages
1. Go to **Messages**
2. Click on a conversation or create a new one
3. Type your message
4. Press Enter or click Send

#### Message Features
- **Emoji Reactions**: Click the emoji icon to react to messages
- **Read Receipts**: See when messages are read (double checkmark)
- **Typing Indicator**: See when someone is typing
- **Online Status**: Know if friends are online

#### Group Chats
1. Click "New Conversation"
2. Select multiple friends
3. Create group chat
4. Start messaging

### Photo Feed

#### Sharing Photos
1. Go to **Feed**
2. Click "Share Photo"
3. Upload image
4. Add caption and location (optional)
5. Click Post

#### Interacting with Photos
- **Like**: Click the heart icon
- **Comment**: Click the comment icon and type
- **Follow**: Click Follow on any user's post

#### Discovering Photos
- View photos from people you follow
- Explore trending photos
- Search by hashtags or location

### Stories

#### Creating Stories
1. Go to **Stories**
2. Click "Create Story"
3. Upload photo or video
4. Add caption (optional)
5. Click Post

#### Viewing Stories
1. Go to **Stories**
2. Click on a story to view
3. Tap next to see more stories
4. Stories disappear after 24 hours

#### Story Analytics
- See who viewed your stories
- Track story engagement
- View story insights

### Contacts & Friends

#### Managing Friends
1. Go to **Contacts**
2. View your friends list
3. See pending requests
4. Accept or reject requests

#### Blocking Users
1. Find the user's profile
2. Click "Block" option
3. Confirm blocking
4. Blocked users can't message you

#### Followers
- See who follows you
- Follow/unfollow users
- View follower lists

### Calls

#### Making Calls
1. Open a conversation
2. Click the call icon
3. Choose voice or video
4. Wait for answer

#### During a Call
- **Mute**: Click microphone icon to mute/unmute
- **Camera**: Click camera icon to toggle video
- **End Call**: Click end call button

#### Call History
- View past calls
- See call duration
- Retry missed calls

### Notifications

#### Notification Types
- **Messages**: New message alerts
- **Friend Requests**: When someone adds you
- **Likes**: When someone likes your post
- **Comments**: When someone comments
- **Story Views**: When someone views your story
- **Follows**: When someone follows you

#### Managing Notifications
1. Go to **Notifications**
2. Click on notification to view
3. Mark as read
4. Delete notification

#### Notification Settings
1. Go to Settings
2. Click Notifications
3. Choose which alerts to receive
4. Set notification frequency

### Search

#### Finding People
1. Go to **Search**
2. Type a name or username
3. Click on result to view profile
4. Send friend request

#### Finding Messages
1. Go to **Search**
2. Select "Messages" tab
3. Type keywords
4. Click to open conversation

#### Finding Photos
1. Go to **Search**
2. Select "Photos" tab
3. Search by caption, hashtag, or user
4. Click to view full photo

## Troubleshooting

### Messages Not Sending
1. Check internet connection
2. Verify recipient is not blocked
3. Try again
4. Contact support if persists

### Photos Not Uploading
1. Check file size (max 50MB)
2. Check file format (JPG, PNG, GIF)
3. Check internet connection
4. Try uploading again

### Can't Find Friends
1. Verify spelling of name
2. Check if user exists
3. Try searching by username
4. Check if you're blocked

### Calls Not Working
1. Check microphone/camera permissions
2. Check internet connection
3. Verify recipient is online
4. Try restarting app

## FAQ

**Q: How long do stories last?**
A: Stories are visible for 24 hours, then automatically deleted.

**Q: Can I edit messages?**
A: Not yet, but this is planned for future updates.

**Q: How do I delete my account?**
A: Go to Settings → Account → Delete Account. This is permanent.

**Q: Is my data encrypted?**
A: Yes, all data is encrypted in transit and at rest.

**Q: Can I use ROCK on multiple devices?**
A: Yes, your account works on all devices simultaneously.

**Q: How do I report a user?**
A: Click the three-dot menu on their profile and select "Report".

**Q: Is ROCK free?**
A: Yes, ROCK is completely free with no ads.
